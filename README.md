# 🧠 Fake Job Predictor

A machine learning-based web application that detects whether a job posting is **real or fake** using Natural Language Processing (NLP) techniques.

---

## 🚀 Features

- Predicts fake vs real job postings
- Uses trained ML model for classification
- Text preprocessing with NLP techniques
- Simple and interactive web interface
- High accuracy prediction based on job description patterns

---

## 🛠️ Tech Stack

- Python
- Flask / Streamlit (for web app)
- Scikit-learn
- Pandas & NumPy
- NLTK / SpaCy
- HTML, CSS (frontend)

---

## 📊 How It Works

1. User enters job description
2. Text is cleaned and preprocessed
3. ML model analyzes patterns
4. Output shows:
   - ✅ Real Job
   - ❌ Fake Job
